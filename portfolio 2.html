PROJECTS

Parkinson’s Disease Classification

This was a thesis project. In this project chellenging 3D images of Parkinson's were preprocessed using Nipype, Nibabel and AntsPY libraries. 3D augmentation techniques were implied using pytorch framework.Developed a custom 3D residual network in order to learn both short-term and long-term memory information from 3D data.

Face Recognition Using Deep Learning for Embedded System

In this project, dlib library was utilized for multi person face recognition. By combining with arduino, for one facial image detection, facenet recognition system was built.

Semantic Analysis of Patent Phrases using Pytorch for Binary Classification

This project was designed to perform semantic analysis in order to find correlation between phrases. Trained a binary classification model to score whether two phrase’s similarity based on the patent class.

Forecasting Beijing’s Air Pollution using Multi-layer LSTM Networks

A time series forecasting model was developed using multi-layer LSTM networks to predict PM2.5 pollution levels of air.Trained the model on historical air quality data from 35 monitoring stations in Beijing, recorded from 2010 to 2015
